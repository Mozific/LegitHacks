# LegitHacks

LegitHacks is a Meteor addon. Disguised as Sodium-Plus.

## Modules
* **LegitReach:** Extended reach with randomized miss logic.
* **LegitAutoFirework:** Automated elytra boosting.
* **LegitChestSwap:** Instant armor/elytra switching.
* **FakeLag:** Packet-based movement delay.
* **KillEffects:** Client-side lightning and sound triggers.
* **StatSpoofer:** Accuracy normalization for anti-cheats.
* **DisconnectSound:** Audio alerts for player logout/range.
* **Panic:** Instant kill-switch for all modules.
